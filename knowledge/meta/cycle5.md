# Cycle 5 — Current State of EVE Frontier

Cycle 5 is the current active development cycle of EVE Frontier, featuring significant improvements to Smart Assemblies, combat, and the player-driven economy.

## What's New in Cycle 5

### Smart Assembly Extensions
Players can now deploy custom Sui Move packages as extensions to their Smart Assemblies. This opens the door to programmable game logic — automated markets, access control, custom services, and more. This is the feature that enables projects like GHOST to exist within the game world.

### Custom dApp URLs
Smart Assemblies now support custom dApp URLs. When a player interacts with an assembly (press F), the game's built-in browser opens the owner's custom web application. This turns every Smart Assembly into a potential service point.

### Improved Combat
Cycle 5 features refined combat mechanics with better Feral AI behavior patterns, new ship modules, and more nuanced damage calculations.

### Builder Scaffold
CCP released the builder-scaffold repository — a Docker-based development environment for building Smart Assembly extensions locally. This includes a local Sui network, pre-deployed world contracts, and example extensions.

### Community Tools
- **efctl**: Community-built CLI for interacting with EVE Frontier's world contracts
- **dApp Kit**: Official toolkit for building in-game and external dApps
- **World Contracts**: Open-source smart contracts that power the game world

## Hackathon Context

The EVE Frontier x Sui Hackathon 2026 celebrates Cycle 5's builder-focused features. The theme "A Toolkit for Civilization" reflects the game's vision: players build the tools, infrastructure, and services that make civilization possible in the void.
